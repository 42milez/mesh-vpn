# mesh-vpn
A decentralized VPN, inspired by [meshbird](https://github.com/meshbird/meshbird).

# Features
- DHT peer discovery
- Port forwarding (NAT-PMP, UPnP, STUN)
- Encryption (AES-256)
- IPv6
